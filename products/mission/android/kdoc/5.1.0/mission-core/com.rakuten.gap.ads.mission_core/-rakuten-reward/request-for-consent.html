Request for Consent if the user have not provide consent.

For application in JAVA code, please use RakutenReward.requestForConsentJava

If user have not provide consent, Consent Dialog will be shown to ask for user's consent. Else the callback will be trigger immediately with RakutenRewardConsentStatus.CONSENT_PROVIDED status.

The consent dialog can only be dismissed when user click on the buttons and the consent status will be returned in the callback.

RakutenReward.requestForConsent { status ->
     if (status == RakutenRewardConsentStatus.CONSENT_PROVIDED) {
         // trigger action when status is CONSENT_PROVIDED
         RakutenReward.logAction("actioncode")
     }
}
