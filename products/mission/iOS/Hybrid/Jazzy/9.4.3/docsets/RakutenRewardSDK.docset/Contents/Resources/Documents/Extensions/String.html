  • dataFromHexadecimalString()

    Create Data from hexadecimal string representation

    This takes a hexadecimal representation and creates a Data object. Note, if the string has any spaces, those are removed. Also if the string started with a ‘<’ or ended with a ‘>’, those are removed, too. This does no validation of the string to ensure it’s a valid hexadecimal string

    The use of strtoul inspired by Martin R at http://stackoverflow.com/a/26284562/1271826

    Declaration

    Swift

    public func dataFromHexadecimalString() -> Data?

    Return Value

    Data represented by this hexadecimal string. Returns nil if string contains characters outside the 0-9 and a-f range.
