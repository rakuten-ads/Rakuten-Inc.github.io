RakutenReward

An object that contains information about Rakuten Reward SDK basic functions, such as SDK initialization, log action, open portal and etc.

  • sharedInstance

    Shared object for RakutenReward.

  • delegate

    Rakuten Reward delegate.

  • isDebug

    Get more logs from Reward SDK by setting isDebug to true. isDebug is false by default.

  • advertisingID

    For iOS 14 and above, please set the IDFA or ad tracking ID to this property.

Public method

  • startSession(appCode:)

    Initialize Reward SDK with appcode.

  • logAction(actionCode:)

    Log action with action code. If you are logging multiple actions at the same time, please use logAction(actionCode:isQueueEnabled:) instead. Logging multiple action without queue may cause some log action to fail.

  • logAction(actionCode:isQueueEnabled:)

    Log action with action code with queue. Use this method when logging multiple actions.

  • openPortal()

    Open webview for Reward SDK portal page.

  • openAdPortal(delegate:)

    Open webview for Reward SDK advertisement portal page.

  • addBlockListURL(urlSet:)

    Add URL to block list to block access to a specific URL. This is for App Store submission rejection. Please use this API if the ad URL has problem.

  • updateMissionList()

    Update user’s mission list (save into user data). This is for internal use only (sample app).

Public getter method

  • getAppCode()

    Application key (appcode) for Reward SDK.

  • getStatus()

    Rakuten Reward SDK status.

  • getUser()

    User information.

  • getVersion()

    SDK version.

  • isOptedOut()

    SDK opt out status.

  • isUIEnabled()

    Mission’s achievement notification UI (Banner, Modal, etc) setting.

Public setter method

  • setUIEnabled(enabled:)

    Set SDK Mission’s achievement notification UI setting. Users is able to change this setting on Reward SDK Portal’s setting page.

Internal method

  • didUpdateLocation(longitude:latitude:timeStamp:)

    This API is deprecated and no longer supported. Will be removed in future version.
