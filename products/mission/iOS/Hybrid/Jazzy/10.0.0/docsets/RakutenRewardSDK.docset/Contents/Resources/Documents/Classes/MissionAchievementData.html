MissionAchievementData

An object that contains information about mission achievement, such as getting the mission’s name, action code, instruction, icon url, notification type etc. You can claim points using ‘claim’ method for achieved mission.

  • init()

    Initialization for mission achievement class, internal use only. Required public access for Unity.

  • claim()

    Claim point(s) for this achieved mission.
